    Connecting to Cloud Servers

    Root user login is disabled by default. Please see instructions on how to connect.
    A Virtual Machine is being provisioned for you, this may take a few moments.
    The cost of the real Cloud Servers are covered as part of your membership.

    How to connect

    1. You can connect to the virtual machine once it is running by selecting Login next to the gear box.

    2. The default username is user and default password is 123456.

    3. On first login you will enter your 123456 password again a second time.

    4. Then you can enter in your new password.
